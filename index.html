Tokenomics

Akrobeto is an interchain token available on multiple blockchain networks, including Base, BNB Chain, and Ethereum. This multi-chain support allows for canonical versions of the token across these blockchains, all sharing a single EVM address. This feature enhances accessibility and enables seamless transactions across platforms. Additional chains can be added based on community demand, ensuring Akrobeto remains adaptable and user-focused.
